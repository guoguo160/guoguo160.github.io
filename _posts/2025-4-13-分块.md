# 分块

## 分块是什么

分块就是把一个数列变成若干个块进行处理，通常来说，块长为 $sqrt(n)$。

## ~~简单~~的一些例题

### [树状数组1](https://www.luogu.com.cn/problem/P3374)

简单分块就行了。如果是在同一个块里面，直接暴力修改就行了。

那么具体怎么操作呢？

显然，我们需要把所有的非整块(以下称为**散块**)进行暴力修改，同时在每一个块内部维护一个 $sum$，查询的时候把所有在区间里面的整块的 $sum$ 加起来就行了。

直接看一下代码：

```cpp
#include <bits/stdc++.h>

#define int long long 

using namespace std;

const int N=5e5+10;

struct Section{
	int st,ed;
	int sum;
};

int n,m; 
int a[N];
int belong[N];
int cnt;
Section sec[N];

signed main() {
	ios::sync_with_stdio(false);
	cin.tie(0);cout.tie(0);
	cin>>n>>m;
	for(int i=1;i<=n;++i) {
		cin>>a[i];
	}	
	int len=sqrt(n);
	for(int i=1;i<=len;i++) {
		sec[i].st=len*(i-1);
		sec[i].ed=sec[i].st+len-1;		
	}
	sec[len].ed=n;
	for(int i=1;i<=len;++i) {
		for(int j=sec[i].st;j<=sec[i].ed;++j) {
			belong[j]=i;
			sec[i].sum+=a[j];
		} 
	}
	while(m--) {
		int opt,x,y;
		cin>>opt>>x>>y;
		if(opt==1) {
			a[x]+=y;
			sec[belong[x]].sum+=y;
		}
		else{
			int sum=0;
			if(belong[x]==belong[y]) {
				for(int i=x;i<=y;++i) {
					sum+=a[i];
				}
				cout<<sum<<"\n";
				continue;
			}
			for(int i=x;i<=sec[belong[x]].ed;++i) {
				sum+=a[i];
			}
			for(int i=belong[x]+1;i<=belong[y]-1;++i) {
				sum+=sec[i].sum;
			} 
			for(int i=sec[belong[y]].st;i<=y;++i) {
				sum+=a[i];
			}
			cout<<sum<<"\n"; 
		}
	}
	return 0;
}

```
